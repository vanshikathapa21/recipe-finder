# 🍲 Khana Khazana – Recipe Finder

A simple and interactive Recipe Finder web application built using React and Node.js.  
Users can search recipes by ingredients, view recipe details, and save their favorite recipes.

---

## 🚀 Features

- 🔍 Search recipes using ingredients
- 📋 View recipe details in a modal
- ❤️ Add recipes to favorites
- ❌ Remove recipes from favorites
- ⭐ Favorites counter in navbar
- 📱 Responsive recipe card layout

---

## 🛠 Tech Stack

Frontend:
- React
- React Router
- CSS

Backend:
- Node.js
- Express.js

API:
- Spoonacular Recipe API
